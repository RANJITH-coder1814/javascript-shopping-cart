# 🛒 JavaScript Shopping Cart App

A beginner-friendly **Shopping Cart Web Application** built using **HTML, CSS, and Vanilla JavaScript**.
This project demonstrates DOM manipulation, event handling, and dynamic price calculation.

---

## 🚀 Features

✔ Add products to cart
✔ Increase / decrease quantity
✔ Remove items from cart
✔ Dynamic total price calculation
✔ Responsive UI
✔ No external libraries used

---

## 🛠️ Technologies Used

* 🌐 HTML5
* 🎨 CSS3
* ⚡ JavaScript (ES6)
